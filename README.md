# 2ezot AJ - Job Monitor

A web-based interface for monitoring and joining jobs from Discord channels.

## Features

- Real-time job monitoring via WebSocket
- Visual job cards with pet information
- One-click job joining
- Activity logging
- Responsive design

## Setup

1. Update the `serverIP` variable in `index.html` to match your Python server's IP
2. Make sure your Python server is running and accessible
3. Open `yourusername.github.io` in a browser

## Usage

- The page will automatically connect to your job monitor server
- Jobs will appear as cards with pet names and earnings
- Click "JOIN JOB" to automatically join in Roblox
- Monitor the activity log for real-time updates

## Requirements

- Your Python job monitor server running on port 1488
- Modern web browser with WebSocket support
- Roblox installed for one-click joining
